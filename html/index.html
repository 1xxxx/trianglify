Trianglify

Trianglify generates colorful geometric patterns from random input. It was inspired by btmills' geopattern project, and can output patterns in a variety of formats. The initial version was written in a single day because I got fed up with Adobe Illustrator. The v0.2.0 release of Trianglify represents a ground-up rewrite which eliminates the dependency on d3.js and includes a brand new canvas rendering implementation.

Features

  • No external dependencies
  • GPL-licensed
  • SVG, PNG, and canvas output
  • Only 32kB minified

Usage

Not a programmer? Check out the Trianglify wallpaper generator example.

Officially maintained packages for Trianglify are on bower, npm, and CDNJS. The examples below are shown for node, but will work with the bower and CDN versions of Trianglify as well (just ignore the require statements).

Trianglify exports a function which takes a single options argument. It returns a pattern object that holds information about the geometry of the generated pattern as well as rendering functions that will render the pattern to an image.

// Basic Trianglify usage
var canvas = Trianglify().canvas(); // Generate a pattern and render it
document.body.appendChild(canvas);  // Append the render to the document
